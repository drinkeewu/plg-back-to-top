# plg-back-to-top
A component make the page back to top

**Install**

```
npm i plg-back-to-top
```



## API

Props

| 属性     | 说明                                        | 类型   | 默认值 |
| -------- | ------------------------------------------- | ------ | ------ |
| height   | 页面滚动高度达到该值时显示**BackToTop**组件 | Number | 200    |
| bottom   | 组件距离容器底部的距离                      | Number | 150    |
| right    | 组件距离容器右部的距离                      | Number | 100    |
| duration | 滚动动画持续时间, 单位 - 毫秒               | Number | 1000   |

Events

| 事件名   | 说明           | 返回值 |
| -------- | -------------- | ------ |
| On-click | 点击按钮时触发 | -      |
